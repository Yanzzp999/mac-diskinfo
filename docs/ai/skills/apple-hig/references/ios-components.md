# iOS Components Reference

## Navigation

- `UINavigationBar` / `NavigationStack`
- `UITabBar` / `TabView`
- `UISearchBar` / `.searchable()`

## Containers

- `UIScrollView` / `ScrollView`
- `UITableView` / `List`
- `UICollectionView` / lazy grids
- `UIStackView` / `VStack`, `HStack`, `ZStack`

## Controls

- `UIButton` / `Button`
- `UISwitch` / `Toggle`
- `UISlider` / `Slider`
- `UISegmentedControl` / segmented picker
- `UITextField` / `TextField`
- `UITextView` / `TextEditor`

## Feedback

- `UIActivityIndicatorView` / `ProgressView`
- `UIAlertController` / `.alert()`
- `.sheet()` and `.fullScreenCover()` for modality
